# PWA icons

The PWA manifest (`/public/manifest.webmanifest`) references the following icons.
Drop the corresponding PNG files in this directory before shipping to production.

| File | Purpose | Recommended size |
|---|---|---|
| `icon-192.png`         | Standard install icon         | 192 × 192 px |
| `icon-512.png`         | Standard install icon (large) | 512 × 512 px |
| `icon-maskable.png`    | Android adaptive icon         | 512 × 512 px (with safe area inside the central 80 %) |
| `apple-touch-icon.png` | iOS home-screen icon          | 180 × 180 px |
| `og-image.png`         | Open Graph share preview      | 1200 × 630 px |
| `favicon-32.png`       | Modern favicon                | 32 × 32 px |

Source asset: `logo2.png` from the maquette can be used as a starting point —
crop with safe area for the maskable variant.

**Brand reference:**
- Primary: `#004e34`
- On-primary: `#ffffff`
- Background: `#f7f9fb`
